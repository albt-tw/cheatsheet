# tmux

## session
    new             -s name
    a               -t name
    kill-session    -t name
    ls

## start with prefix

### session
    s    session list
    (    previous session
    )    next session

### window
    w    window list
    c    create window
    ,    name window
    .    move window
    &    kill window

### panel
    !    panel to new tab
    x    kill panel
    {    move panel left
    }    move panel right
